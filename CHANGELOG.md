All notable changes to Manufacturing Vision Analyzer will be documented in this file.

and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-02-10

### Added
- Complete .NET 10 console application
- Azure AI Vision integration (Caption, Tags, Objects)
- Cross-modal knowledge graph architecture
- Interactive 13-option menu system
- Analytics dashboard with multiple visualizations
- Cross-product pattern discovery feature
- Equipment recommendations system
- Console-based charts (bar, pie, heatmap, network)
- JSON persistence for knowledge graph
- Dashboard export to timestamped text files
- MVTec Anomaly Detection dataset support
- Domain knowledge integration (ISO 9001 standards)
- Rate limiting for Azure API calls (Free tier compatible)

### Features
- Query defects by product type
- Find similar defects across products
- Custom search by defect type
- View all products in database
- Generate visual diagrams
- Export results to file
- Sample insights display
- Cache management (save, rebuild, delete)

### Technical
- .NET 10.0 target framework
- Azure.AI.Vision.ImageAnalysis 1.0.0-beta.3
- In-memory graph data structure
- 15 product categories support
- Real-time AI-generated insights

## [1.0.0] - Initial Release

### Added
- Basic knowledge graph functionality
- Azure Vision API integration
- MVTec dataset processing
- Simple query interface
